## 📦 Quick Start

```bash
# Clone repository
git clone https://github.com/karamik/AUGUR.git
cd AUGUR

# Backend setup
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload

# Frontend setup
cd frontend
npm install
npm start
```

Access: http://localhost:3000
